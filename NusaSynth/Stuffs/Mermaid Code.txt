    BG["<b>Background Research</b>
    ─────────────────────
    • Dataset NusaX-Senti hanya memiliki 500 data train per bahasa daerah
    • Kurasi manual oleh penutur asli mahal, lambat, dan tidak scalable
    • LLM modern bias terhadap bahasa Inggris dan budaya Barat"] --> PR["<b>Problems</b>
    ─────────────────────
    • LLM tunggal rentan menghasilkan campur kode dan halusinasi linguistik
    • Tidak ada mekanisme validasi otomatis untuk kemurnian bahasa dan konsistensi label sentimen
    • Belum ada framework multi-agent untuk generasi data sentimen bahasa daerah Indonesia"]
    PR --> ST["<b>Study</b>
    ─────────────────────
    • Pengembangan NusaSynth, framework
    • Multi-Agent untuk generasi data sentimen sintetis pada bahasa daerah low-resource"]
    ST --> CT["<b>Contribution</b>
    ─────────────────────
    • Arsitektur multi-agen spesialis Contextualizer, Generator, Sentiment Validator, Linguistic Validator
    • Mekanisme feedback loop untuk kemurnian linguistik dan konsistensi label sentimen
    • Augmentasi dataset NusaX-Senti berbasis Agentic AI"]
    CT --> IN["<b>Insight</b>
    ─────────────────────
    • Peningkatan F1-Score pada downstream task IndoBERT/XLM-R
    • Data sintetis bebas campur kode dengan label sentimen konsisten
    • Framework yang scalable dan replicable untuk bahasa low-resource lainnya"]
